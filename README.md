# Portfólio Profissional - Matheus Barbosa

## Sobre o Projeto

Portfólio profissional desenvolvido para apresentar meus projetos e experiências como Full Stack Developer.

## Como executar localmente

Requisitos: Node.js & npm instalados - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Siga os passos:

```sh
# Passo 1: Clone o repositório
git clone https://mhbapp.netlify.app

# Passo 2: Entre na pasta do projeto
cd Portifolio

# Passo 3: Instale as dependências
npm i

# Passo 4: Inicie o servidor de desenvolvimento
npm run dev
```
